#### 2. Gather the Logos

```sh
deposit=".assets" && mkdir -p "$deposit"
baseurl="https://github.com/olankens/logomain/raw/HEAD/source"
for f in {angular,chromium,intellijidea,kotlin,spring,kubernetes}; do
  curl -Lo "$deposit/$f.svg" "$baseurl/$f.svg"
done
```
